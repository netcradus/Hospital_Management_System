# Hospital Management System

A monorepo scaffold for a full-stack hospital management platform with a React frontend and an Express backend.

## Structure

- `client/` React + Vite + Tailwind starter
- `server/` Express + MongoDB API starter

## Quick Start

```bash
cd server && npm install
cd ../client && npm install
```

Copy env files:

```bash
cp server/.env.example server/.env.local
cp client/.env.example client/.env.local
```

Run the backend with `npm run dev` in `server/`, and the frontend with `npm run dev` in `client/`.
